# Iniciacao-Cientifica
# Análise de Insuficiência Cardíaca via Formal Concept Analysis (FCA)

Este repositório contém o desenvolvimento de uma pesquisa de **Iniciação Científica** que utiliza técnicas de Ciência da Computação para analisar dados médicos. O foco central é a aplicação da **Análise Formal de Conceitos (FCA)** para identificar perfis de risco em pacientes com insuficiência cardíaca.

## 🚀 Sobre o Projeto
O objetivo deste trabalho é transformar dados clínicos numéricos em um **Contexto Formal Diádico**, permitindo a geração de um **Reticulado de Conceitos (Concept Lattice)**. Através dessa estrutura, buscamos identificar associações entre variáveis como Fração de Ejeção, Creatinina Sérica e a taxa de sobrevivência dos pacientes.

A base de dados utilizada é composta por registros de 299 pacientes do Paquistão, focando nos preditores identificados pela literatura médica (Chicco & Jurman, 2020) como os mais relevantes para o prognóstico da doença.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Python (Pandas para processamento e discretização).
- **FCA:** Lattice Miner / ConExp (para visualização de diagramas de Hasse).
- **Formatação:** LaTeX (para documentação científica).
